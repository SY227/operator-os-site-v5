# Legacy Content Archive (V5)

This folder contains JSON content files carried over from earlier Operator OS / V2-V4 architectures.

These files are **not used** by active V5 public routes.

Archived files:
- articles.json
- products.json
- free-tools.json
- playbooks.json
- templates.json
- testimonials.json
- diagnostic-questions.json

Why archived instead of deleted:
- preserves historical copy and structure for reference
- avoids losing migration context
- keeps active `/content` model clean for OpenClaw operations

Active V5 content files remain at `/content` root:
- site-config.json
- blog-posts.json
- agent-projects.json
- use-cases.json
- chatbot-faq.json
